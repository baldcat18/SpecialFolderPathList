・名前: SpecialFolderPathList.hta
・作者: BaldCat

・概要
Windows で使われている特殊フォルダーのフルパスを表示します。
パスをクリックすると、そのフォルダーを開きます。
(一部のフォルダーは開くのに管理者権限が必要です。)

[コントロール パネル] などの仮想フォルダーも扱っています。

パスを右クリックすると、コンテキストメニューが表示されます。
フォルダーのパスをクリップボードにコピー、
コマンドプロンプトや PowerShell の起動、
フォルダーのプロパティの表示ができます。

・動作環境
Windows 8.1 Update
Windows 10 バージョン 1903、1909、2004、20H2

32 ビット版・64 ビット版の両方で動作します。

Windows 7 SP1 や Windows 10 バージョン 1809 でも動作すると思いますが、動作確認していません。

Windows 7 で実行するには Internet Explorer 11 が必要です。

Windows RT、Windows 10 S などの HTA をサポートしない環境では動作しません。

・開発環境
Visual Studio Code

・再配布・改変について
中身を改変しない場合の再配布に制限はありません。
改変したものを配布する場合は、名前を変えていただけるとありがたいです。
(「○○版 SpecialFolderPathList」などのような形でも
 完全なオリジナル名でも結構です。)
